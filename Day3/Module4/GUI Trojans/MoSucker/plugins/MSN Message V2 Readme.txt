 ------------------ Description of the features: ----------------------
This plugin will silently send a message to all online MSN Messenger
contacts. There will be no indication on the victim's computer that
a message has been sent.

You may also retrieve all contacts on the victim's MSN online list.
 -----------------------------------------------------------------------
 Note: Plugin will cause an object error if MSN messenger has not been
       installed on the computer. Check the process manager for msmsgs.exe.

 How to use:
 Copy the files "MSN Message.gui" and "msnmsg.msp" to your
 MoSucker\plugins directory.
 Start the client, select "Plugins" in the "Misc stuff"-menu
 and doubleclick the "MSN Message".

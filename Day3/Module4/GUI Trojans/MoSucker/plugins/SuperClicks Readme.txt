 Plugin file:          superclicks.msp
 Plugin size:          16KB

 ------------------ Description of the features: ----------------------
This plugin will cause an image to appear everywhere the victim clicks
for a defined number of mouseclicks. You may select from several different
images. View superclicks.gif for an image preview.
 -----------------------------------------------------------------------


 How to use:
 Copy the files "SuperClicks.gui" and "superclicks.msp" to your
 MoSucker\plugins directory.
 Start the client, select "Plugins" in the "Misc stuff"-menu
 and doubleclick the "SuperClicks".
